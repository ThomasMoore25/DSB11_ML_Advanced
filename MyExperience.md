# DSB11: Advanced Machine Learning

## О чём проект?

Продвинутые методы ML: регуляризация (L1/L2/ElasticNet), ансамбли (Bagging/Boosting/Stacking), кластеризация, снижение размерности (PCA, t-SNE).

## Технологии и инструменты

- Python 3
- scikit-learn
- XGBoost
- LightGBM
- CatBoost
- pandas
- numpy

## Чему научился

- Регуляризация: Ridge, Lasso, ElasticNet — влияние на коэффициенты
- Ансамбли: BaggingClassifier, RandomForest, GradientBoosting
- Boosting: XGBoost, LightGBM, CatBoost — сравнение производительности
- Кластеризация: KMeans, DBSCAN, иерархическая
- Снижение размерности: PCA, t-SNE, UMAP

## Моя часть в проекте

Индивидуальный проект: реализовал и сравнил продвинутые модели на датасете с регуляризацией.
